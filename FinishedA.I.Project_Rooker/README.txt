The program uses the communicative exploration aglgorithm from the Rooker article. 
When starting the program it will ask for the desired number of robots and the range.
Then a random evironment will be genrated with random starting spots for the robots. 
Attached to this program is also 3 video recording of the tests being done for the following:
	5 robots with 20 range
	4 robots with 15 range
	3 robots with 10 range
These videos have been sped up to show the results faster.
